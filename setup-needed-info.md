# 需要填入的信息

已填入：

- Supabase Project URL
- Supabase publishable / anon public key

已填入：

- 你的登录邮箱
- 你女朋友的登录邮箱

## 获取位置

- Supabase Project URL：Supabase 项目后台 -> Project Settings -> API -> Project URL
- Supabase anon public key：Supabase 项目后台 -> Project Settings -> API -> Project API keys -> anon public
